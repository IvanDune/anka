# Где
В стороне гор ( название?) старинный вулкан который заснул сотни лет назад в его жерле появилось озеро их дождевой воды и подводных источников. Озеро лунных слез. вокруг этого озера словно изъеденная термитами кора - туннели гномов.

## Структура
 Изгнав гномов ( из-за этого плохие взаимоотношения с ними) темные эльфы укрылись под землей. 
 Правление - совет Старейшин - представители каждого рода собираются каждую четверть [[Лунная четверть]]
